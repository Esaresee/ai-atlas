# The AI Productivity Atlas

A premium, interactive guide to the best AI tools of 2025. This single-file application functions like a digital book or dashboard, allowing users to explore, track, and master the AI landscape.

## Features

- **Interactive Atlas**: Navigate through 8 core categories of AI tools.
- **"My Stack" Tracking**: Check off the tools you use. Your choices are saved automatically to your browser (Local Storage), so they are remembered when you return.
- **Real-Time Search**: Instantly filter the Atlas to find specific tools or categories.
- **Dark Mode**: A toggleable, OLED-friendly dark theme that persists across sessions.
- **Pricing Filters**: Easily toggle to show only Free/Freemium tools in any chapter.
- **Deep Dives**: Detailed chapters with "Best Practices", "Warnings", and direct links to official websites.

## How to Use

1.  Ensure `interactive_guide.html` and `cover.png` are in the same folder.
2.  Open `interactive_guide.html` in any modern web browser (Chrome, Edge, Safari, Firefox).
3.  No internet connection is required for the content (though links to tools will need internet).

## Deployment

To share this guide with others, you can host it for free:

- **GitHub Pages**: Upload to a repository and enable Pages.
- **Netlify Drop**: Drag and drop the folder to Netlify.
